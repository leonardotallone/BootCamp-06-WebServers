Web Server

El termino Server puede hacer referencia tanto a hardware como software, en esta ocacion haremos referencia a esta última.

Un server es el programa que nos permite controlar como un usuario final puede consume información, principalmente archivos estaticos. Estos archivos suelen agruparse para formar una pagina web.

El principal medio de comunicacion es via el protocolo HTTP, cada vez que un browser busca un archivo, debera generar un pedido HTTP hacia el servidor para conseguirlo.
